

# Plan: Telegram-уведомления о покупке билетов

## Что будет сделано

При каждой успешной верификации оплаты (в `verify-payment`) система будет отправлять вам сообщение в Telegram с информацией о покупке: имя гостя, мероприятие, сумма, количество мест, номер билета.

## Шаги

### 1. Подключить Telegram-коннектор
Подключим Telegram-бота через встроенный коннектор Lovable. Вам нужно будет создать бота через @BotFather в Telegram (если ещё нет) и подключить его.

### 2. Узнать ваш chat_id
Нужно будет узнать ваш личный chat_id в Telegram, чтобы бот знал, куда отправлять уведомления. Это можно сделать, написав боту `/start` и получив chat_id через API.

### 3. Добавить отправку уведомления в `verify-payment`
В Edge Function `verify-payment/index.ts`, после успешной верификации оплаты (строка ~489, после создания билета), добавим вызов Telegram Bot API через connector gateway:

```
📩 Новая покупка билета!
👤 Гость: Сергей Черненко
🎫 Билет: ABC123DEF456
🎭 Мероприятие: Стратегия наследия
📅 Дата: 16 мая 2026
💰 Сумма: 25 000 ₽
🪑 Мест: 1
```

### 4. Передеплоить функцию
После изменений — автоматический деплой `verify-payment`.

## Технические детали

- Используется Telegram connector gateway (`connector-gateway.lovable.dev/telegram/sendMessage`)
- Секреты `LOVABLE_API_KEY` и `TELEGRAM_API_KEY` будут доступны автоматически после подключения коннектора
- Chat ID будет сохранён как константа в Edge Function
- Отправка уведомления не блокирует ответ пользователю — ошибка Telegram логируется, но не влияет на процесс покупки
